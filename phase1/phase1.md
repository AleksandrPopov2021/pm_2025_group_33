# Фаза 1: Выбор концепции проекта
## Шаг 1: Придумать не менее 4 концепций программного проекта
В рамках темы были предложены 4 концепции продуктов:
1. **Сервис планирования учёбы для студентов** - Веб-приложение, которое помогает студентам планировать расписание занятий, дедлайны и напоминает о них. Поддержка синхронизации с Google Calendar и Telegram-ботом для уведомлений.
2. **Мобильное приложение для отслеживания настроения и самочувствия** - Пользователь ежедневно оценивает настроение, сон, уровень стресса. Приложение визуализирует динамику и даёт рекомендации (например, “вы мало спали последние 3 дня”).
3. **Онлайн-платформа для совместного решения задач по программированию** - Аналог LeetCode с возможностью командной работы в реальном времени, чатом и обсуждением решений.
4. **Генератор персональных тренировок без оборудования** - Приложение создаёт индивидуальные планы тренировок на основе уровня подготовки, целей и доступного времени. Поддерживает прогресс-трекинг.
